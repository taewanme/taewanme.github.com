    OCI Livelogs

    OCI Livelogs

    OCI Livelogs는 오라클 클라우드(OCI, Oracle Cloud Infrastructure) 사용법을 데모 시나리오 중심의 영상으로 소개하는 콘텐츠입니다. 오라클 클라우드에서 특정 작업을 수행하는 시나리오를 소개하고 그 과정을 영상으로 제시함으로써 직관적인 정보 전달을 목표로 합니다. 이미지와 문장 중심의 매뉴얼은 작업 과정을 상상해야 하는 불편함과 어려움이 있습니다. 매뉴얼을 보고 실습을 하기 위해서는 작업 환경 준비 및 은근히 복잡한 과정이 필요합니다. OCI Livelgos 실습 영상을 통해서 정보 전달력과 직관적인 이해를 높이고, 사용자가 실습을 따라 하는 부담을 최소화하는 것이 OCI LiveLogs의 지향점입니다.

    Oracle Cloud 실습 문서 유형

    오라클 클라우드는 시나리오 중심으로 클라우드 사용법을 쉽게 소개하는 다양한 문서를 제공합니다.

    • Oracle Livelabs
    • OCI Develper Tutorials
    • Oracle Architecture Center

    각 문서의 특징은 다음과 같이 요약할 수 있습니다.

    Oracle Livelabs

    Oracle Livelabs는 시나리오 형 오라클 클라우드 실습 매뉴얼입니다. Oracle Livelabs은 이미지 중심의 단계별(Step-by-Step) 따라하기 형식으로 작성된 실습 온라인 문서입니다. 단편적인 실습 보다는 종합적인 시나리오를 제시하고, 제시한 주제를 해결하는 과정을 소개하는 형식으로 구성됩니다. 2021년 9월 18일 기준 255개의 OCI 실습 워크샵 문서를 제공합니다. Oracle Livelabs는 다음 URL에서 목록을 확인할 수 있습니다.

    • https://apexapps.oracle.com/pls/apex/dbpm/r/livelabs/livelabs-workshop-cards

    OCI Develper Tutorials

    입문자를 위한 개발자 튜토리얼은 OCI 공식 문서의 일부분으로 입문자를 위한 8개 서비스(Free Tier, Functions, Kubernetes, Terraform, Data Catalog, Data Flow, Data Integration, Data Science)에 대한 초보자 실습 문서 42개 (2021년 9월 18일 기준 )를 제공합니다. 이 문서는 이미지 보다는 텍스트를 중심으로 작성됩니다. 입문자를 위한 개발자 튜토리얼은 다음 URL에서 확인할 수 있습니다.

    https://docs.oracle.com/en-us/iaas/developer-tutorials/tutorials/home.htm

    Oracle Architecture Center

    오라클 클라우드 서비스를 이용하여 특정 엔터프라이즈 이슈를 해결하는 아키텍처 및 구축 방법, 구현체를 소개하는 문서 353개(2021년 9월 18일 기준 )를 제공합니다. 아키텍처 이슈 소개, 이슈 해결을 위한 아키텍처 제시, Terraform으로 솔루션 배포로 구성됩니다. 오라클 아키텍처 센터에서 제공하는 문서 목록은 다음 URL에서 확인할 수 있습니다.

    • https://docs.oracle.com/solutions/

    OCI Livelogs

    Oracle Livelabs, Developer Tutorial 및 아키텍처 센터는 다음과 같은 문제점을 갖고 있습니다.

    • 한글화 부족
    • 이미지와 문자 중심의 문서

    OCI Livelogs는 위 3개 문서를 대상으로 한글화하고 실습과정을 영상으로 녹화하여, 매뉴얼 내용과 진행 과정을 영상으로 확인할 수 있는 콘텐츠입니다. OCI Livelogs는 한글화된 문서와 Youtube 실습 동영상으로 구성됩니다. OCI Livelog를 통해서 다음과 같은 효과를 얻을 수 있습니다.

    • 오라클 클라우드 실습형 메뉴얼 한글화
    • 실습 과정을 영상으로 사전 확인
    • 사용자가 실습을 진행할 경우 가이드라인 역할

    오라클 클라우드에 대한 정보가 필요한 사용자가 오라클 클라우드 매뉴얼 습득의 효율성을 높이고 직과적으로 기술 이해를 돕는 콘텐츠를 양산하는 것이 OCI Livelogs의 지향점입니다.
