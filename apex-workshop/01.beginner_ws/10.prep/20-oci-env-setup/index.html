    OCI 기본 환경 설정

    • 영상 재생 시간: 14:18

    Oracle APEX 입문 워크숖 실습의 통일성을 높이기 위하여 OCI 기본 환경 다음과 같이 구성하겠습니다.

    OCI 사용자 ID & 사용자 그룹

    다음과 같은 사용자 ID를 OCI IAM으로 만들고, OCI 사용자 그룹에 설정하겠습니다.

    IAM User ID IAM USER Group 설명
    admin administrators OCI 계정 관리자 ID
    developer01 develoeprs 개발자 ID

    OCI Compartment

    OCI Compartment(구획)은 OCI 자원을 논리적으로 묶는 OCI IAM 핵심 구성 요소입니다. OCI Compartment(구획) 계층 구조로 만들어집니다. 다음과 같은 계층 구조로 apexws와 livelabs 컴파트먼트를 만들겠습니다. 이번 워크샾의 모든 자원은 projects/apexws 컴파트먼트에 생성합니다.

    • PROJECT
      • APEXWS
      • LIVELABS

    OCI Policy

    root 컴파트먼트에 다음과 같은 policy를 생성합니다.

    Allow group developers to manage all-resources in compartment PROJECT:APEXWS
