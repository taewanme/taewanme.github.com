Oracle MCD 2019 발표문서: Cloud Native Java: GraalVM

November 17, 2019 graalvm

2019년 11월 19일 Oracle MCD(Modern Cloud Day) Seoul 2019에서 발표한 GraalVM 발표 문서입니다. GraalVM에 대한 전반적인 소개와 AOT Native Image에 대해서 다룹니다. GraalVM이 지향하는 방향을 소개하는 입문자용 문서입니다. 마지막 부분에 Native Image의 PGO(“Profile-Guided Optimization)과 grGraal에 대한 내용을 다룹니다.
