JShell 사용자 가이드(번역)

December 27, 2017
최근에 각광받는 여러 언어의 특징은 코드 실행과 테스트가 쉽다는 것입니다. 이런 기능은 주로 REPL 기능으로 제공됩니다. 자바는 컴파일 언어이기 때문에 이러한 기능 제공이 부족했스빈다. Java 9에 REPL 기능이 추가되었습니다. 자바 REPL 기능의 이름이 Java Shell입니다. Java 9에서 새로 소개된 Java Shell의 공식 사용자 가이드를 번역해 보았습니다. 총 7개 문서로 구성되어 있습니다. Java Shell의 구요 기능, 각 기능의 사용법과 응용법을 다룹니다. .
