Oracle Database의 scott/hr 계정 활성화

November 27, 2015

Oracle Database은 데모 계정으로 scott와 hr을 제공합니다. 이 두 계정은 기본적으로 lock 상태로 설치됩니다. 두 계정을 활성화 시키는 명령은 다음과 같습니다.

System 계정으로 다음 명령을 수행합니다.

alter user scott identified by tiger account unlock;
alter user hr identified by hr account unlock;

위 두 명령은 scott와 hr 계정을 활성화 시킵니다. 두 계정의 로그인 패스워드는 각각 tiger와 hr로 설정됩니다.

scott / tiger에 자원 사용 권한 설정을 해야 할 경우 설정 명령은 다음과 같습니다. 다음 명령은 system 계정으로 수행합니다.

GRANT CONNECT,RESOURCE,UNLIMITED TABLESPACE TO SCOTT IDENTIFIED BY TIGER;
GRANT CONNECT,RESOURCE,UNLIMITED TABLESPACE TO HR IDENTIFIED BY  HR;
