오라클 클라우드 서비스 공식 로고 이미지 모음: 2021.04

October 14, 2021 Cloud

2021년 4월 오라클에서 배포한 OCI(Oracle Cloud Infrastructure, 오라클 킄라우드) 공식 서비스 로고 이미지를 정리했습니다. 2021년 10월 현재 공식 OCI 서비스 공식 로고 이미지의 최신 버전입니다. 이 페이지를 통해서 OCI 공식 로고를 찾는 수고를 덜었으면 합니다. 향후 문서화를 진행할 때 정확한 정보 전달을 위하여 이 로고를 사용하여 문서화하고 컨텐츠를 제작하겠습니다.

전체 이미지는 다음 링크를 통해서 다운로드 받으실 수 있습니다.
