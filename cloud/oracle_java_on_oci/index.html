Oracle OCI에서 'Oracle Java SE is free'

April 19, 2019 cloud

2019년 4월 10일에 “OTN License Agreement for Oracle Java SE“가 업데이트되었습니다. 이번 업데이트에서 주목할 점은 Oracle OCI(Oracle Cloud Infrastructure, 오라클 클라우드)에서 Oracle Java SE 사용에 대한 구체적인 언급입니다.

이번 라이센스 업데이트는 Oracle OCI에서 Oracle Java SE의 Subscription 없이 자유로이 사용할 수 있음을 명시하고 있습니다.

다음 이미지는 Oracle Technology Network License Agreement for Oracle Java SE[↗NW] 에서 발취한 내용입니다.

  • 그림 1. “OTN License Agreement for Oracle Java SE”의 OCI User 관련 부분

위 문서에서 Oracle Java SE는 다음 4가지 경우에 Oracle Java SE를 무료로 사용할 수 있다고 정리하고 있습니다.

  • Desktop과 Notebook에서 개인적인 목적으로 사용 (게임, 애플리케이션 사용 등 JRE로 사용하는 경우)
  • 개발, 테스트, 프로토 타임, 디버깅, 프로파일링, 빌드 목적으로 사용하는 경우
    • 운영 상태가 아닌 개발, 빌드, 테스트 용도의 사용은 제약이 없습니다.
  • 오라클이 승인하는 소프트웨어에 bundle 된 Oracle Java SE의 사용
  • 오라클 클라우드인 OCI에서 Oracle Java SE를 사용하는 경우1
