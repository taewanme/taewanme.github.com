새로운 Oracle Cloud 프로모션: Free Tier

September 24, 2019 cloud

2019년 9월 16일부터 19일까지 4일간 샌프란시스코에서 Oracle Open World 2019(이하 OOW 2019)가 진행됐습니다. 이번 OOW 2019 첫째 날 기조연설에서 오라클의 CTO 래리 엘리슨은 오라클 클라우드 주요 전략을 발표했습니다.

그 중에서 가장 큰 주목을 받은 부분은 다음의 두 가지입니다.

  • 현재 16개인 클라우드 데이터센터(리전)를 2020년까지 36개로 확대
  • 새로운 오라클 클라우드 프로모션 Free Tier 공개

래리 엘리슨이 발표한 클라우드 데이터센터 증설 계획에는 한국의 두 번째 구축 계획이 포함되어 있어 많은 관심을 받았습니다. 내년에 한국 두 번째 오라클 데이터 센터를 춘천에 오픈할 계획을 발표했습니다. 그리고 Oracle Cloud(OCI)와 MS Azure의 센터 간 연동을 한국에 구성할 계획도 알렸습니다.

또 한 가지 주목받은 부분은 Oracle Cloud의 새로운 프로모션 프로그램인 Oracle Cloud Free Tier입니다. Oracle Cloud Free Tier[↗NW] 는 기존에 30일 동안 300달러 상당의 오라클 클라우드 서비스 무료 사용권을 제공하던 Oracle Cloud Free Trial을 확대한 새로운 사용자 지원 계획입니다.

9월 17일 “매일 경제”는 이번에 발표한 “Oracle Cloud Free Tier“에 대해서 다음과 같이 보도 했습니다.

오라클이 아마존웹서비스, 마이크로소프트(MS) 등이 주름잡고 있는 기업용 클라우드 서비스 시장에 선전포고를 던졌다. 일단 사용자 한 명당 연간 3,500만원에 달하는 서비스를 기간 제한 없이 무료로 제공한다는 방침을 밝혔다. (출처: https://www.mk.co.kr/news/it/view/2019/09/738006/[↗NW] )

이 두 가지 이슈 중에서 새로 공개한 “Oracle Cloud Free Tier” 프로모션에 대해서 정리하겠습니다. 이 문서에서는 다음과 같은 내용을 다룹니다.

  • Oracle Cloud Free Tier 소개
    • 구성 컴포넌트: 1. Free Trial
    • 구성 컴포넌트: 2. Always Free
  • Oracle Cloud Free Trial 계정 신청 절차

Free Tier를 등록하시는 분들은 혹시 모를 “요금 폭탄“에 대해서 걱정하십니다. Oracle Cloud Free Tier 계정에는 자원 할당 한계가 설정되어 있습니다. Oracle Cloud Free Tier 계정은 허용된 자원을 초과해서 사용할 수 없습니다. Oracle Cloud Free Tier 계정이 할당된 자원 이상의 서비스를 사용하고자 한다면 계정 업그레이드 절차가 필요합니다. Oracle Cloud Free Tier 계정에서는 별도 비용이 전혀 발생하지 않습니다.

Oracle Cloud Free Tier

Oracle Cloud Free Tier는 오라클 클라우드 경험 기회를 사용자에게 제공하기 위한 프로그램입니다. Oracle Cloud Free Tier는 다음과 같은 2개 프로모션으로 구성됩니다.

  • Oracle Cloud Free Trial
  • Oracle Cloud Always Free

“Oracle Cloud Free Trial“은 오라클 클라우드 계정을 만들면 누구나 30일 동안 300달러에 해당하는 클라우드 자원을 자유롭게 사용할 수 있는 프로그램입니다. “Oracle Cloud Always Free“는 특정 클라우드 서비스를 기간과 비용에 제약 없이 사용할 수 있는 프로그램입니다.

오라클은 Oracle Cloud Free Trial 프로그램을 3년 전부터 운영해 왔습니다. Oracle Cloud Free Tier는 기존 Oracle Cloud Free Trial에 Oracle Cloud Always Free를 추가한 모델입니다.

2019년 9월 16일부터 Oracle Cloud Free Tier 계정으로 Oracle Cloud Free Trial과 Oracle Cloud Always Free를 모두 이용할 수 있습니다.

1. Oracle Cloud Free Trial

이 프로그램은 오라클 클라우드를 30일 동안 300달러 규모로 클라우드 서비스를 사용하는 프로모션입니다. 사용자가 오라클 클라우드 서비스를 사용하면 해당 서비스 사용 시간을 기준으로 이용 금액이 차감됩니다. 일반적으로 PaaS 서비스(Autonomous Service 등) 사용 비용은 IaaS 컴퓨팅 서비스 보다 높습니다. “Oracle Cloud Free Trial” 사용 기간에 모든 PaaS 서비스는 IaaS 컴퓨팅 가격으로 계산됩니다. 따라서 300달러로 상당히 많은 서비스를 이용할 수 있습니다.

“Oracle Cloud Free Trial“이 지원하는 주요 서비스는 다음과 같습니다.

  • Infrasturcture
    • Compute: 3,500시간, 1.5TB 스토리지, VM과 베어메탈(Bare Metal) 서버
    • Storage: 5TB Object Storage
    • Container: 관리형 쿠버네티스 (컴퓨트 기준 3,500시간), 2TB 규모의 Docker Registry
    • Function: 서버리스 서비스로 1억 1천 2백만번 실행
    • Load Balancer: 3500시간
  • Database
    • Autonomous Transaction Processing(ATP)
    • Autonomous Data Warehouse(ADW)
    • NoSQL Database: 10억번 쓰기(1.25 billion writes), 25억번 읽기(2.5 billion reads), 저장공강 100GB
    • Oracle Database Cluod Service: 3,200 시간
  • Application Development
    • Java: 관리형 Weblogic 서비스 1,700 시간
    • Digital Assistant: 챗봇 서비스, 110,000 요청 처리
    • Mobile Hub: 110,000 요청 처리
    • Messaging: 8억개 메시지 처리
  • Analytics
    • Oracle Analytics: 4,700 시간

위 목록에서 제시된 시간은 각 서비스를 구성하는 전체 VM 서버 실행 시간을 합친 값입니다. 예를 들어서 관리형 Kubernetes 서비스(OKE)의 클러스터 인스턴스를 3개 VM으로 구성하고 10시간 사용하면, 이 서비스의 총 사용 시간은 30시간으로 계산됩니다.

2. Oracle Cloud Always Free

이제 Oracle Cloud Free Tier 계정을 등록한 사용자는 Oracle Cloud Always Free 프로그램을 통해서 다음과 같은 서비스를 기간 제약없이 계속 이용할 수 있습니다.

Oracle Cloud Free Tier 계정 등록

2019년 9월 16일 Oracle Cloud Free Tier를 새로 공개하면서 Oracle Cloud 프로모션 계정 등록 절차가 일부 변경되었습니다. Oracle Cloud Free Tier는 사용자별로 1회 제공되는 프로모션입니다. 따라서 다음과 같은 정보가 필요합니다.

  • 개인 메일 (gmail, naver, daum 메일 가능)
  • 전화 번호
  • 신용 카드 번호

계정을 등록할 때 입력한 개인 메일과 전화번호는 기존에 Oracle Cloud Free Tier(혹은 이전 Free Trial) 등록에 사용된 이력이 있는지 검증합니다. 따라서 이전에 사용 이력이 없는 메일과 전화 번호를 사용해야 합니다. 신용카드는 일정 횟수 중복 사용을 허용합니다. 그러나 가능한 기존에 사용하지 않는 새로운 카드 사용을 권장합니다. 새로 발급한 은행 직불카드(비자, 마스터)를 사용하는 것이 편리합니다.

Oracle Cloud Free Tier 계정 등록에 사용되는 신용 카드 정보는 인증 용도로만 사용되고 별도의 과금에는 이용되지 않습니다. 여기서 등록된 카드 정보는 사용자 인증을 위하여 1불 결재를 시도하고 바로 결재를 취소하는 계정 등록 프로세스에 1회만 사용됩니다.

Free Tier 계정 등록 절차

Oracle Cloud Free Tier 등록 페이지는 https://oracle.com[↗NW] 에서 찾을 수 있습니다. 다음 이미지와 같이 oracle.com 홈페이지에서 Oracle Cloud Free Tier 메뉴로 Free Tier 메인 페이지에 이동할 수 있습니다.

Oracle Cloud Free Tier 메인 페이지에서 “Start for free” 버튼을 클릭하면 계정 등록 페이지로 이동합니다.

현재 사용하는 브라우저로 오라클 클라우드에 처음 접속할 경우에 쿠키 정보 수집과 관련한 사용자 동의 페이지가 출력될 수 있습니다. 쿠키 정보 수집을 동의하면 계정 등록 페이지로 이동합니다.

이제 계정 등록 절차가 시작됩니다. 계정 등록에 사용할 E-mail 주소와 현재 위치 및 사용자 정보 사용에 대한 동의를 체크하고 “Next” 버튼을 클릭하여 다음으로 이동합니다. 아래 페이지에서 사용 가능한 E-mail인지 검증이 진행됩니다.

다음 페이지에서 사용 목적, 홈 리전 위치, 등록자 이름, 주소, 소속 국가 및 전화번호 입력이 진행됩니다. 주소를 입력할 때, 다섯 자리 우편 번호를 사용하는 신주소 체계를 사용합니다. 전화번호를 입력할 경우에 첫 번째 번호는 생략해야 합니다. 예를 들어서 010-1234-1234일 경우에 국가 코드를 제외하고 10-1234-1234로 입력해야 합니다. “Next:Verify Mobile Number” 버튼을 클릭하면 주소 및 전화번호 확인 절차로 넘어갑니다.

앞에서 입력한 주소 확인 팝업 창이 오픈됩니다. 잘 못 입력되었으면 “Edit“으로 수정하고 맞으면 “Use Original“를 클릭합니다.

위 이미지에서 주소를 확인하면, 앞에서 입력한 전화번호로 다음과 같은 SMS 문자가 전송됩니다. SMS 문자에는 여섯 자리 인증 코드가 포함됩니다.

SMS 포함된 인증 코드를 다음 팝업 창에 입력하고 “Verify Code” 버튼을 클릭합니다. 인승 번호가 맞으면 패스워드 설정 페이지로 이동합니다.

다음과 같은 규칙으로 패스워드를 입력하고 “Next: Payment Information” 버튼을 클릭하고 신용카드 등록 페이지로 이동합니다.

다음 이미지와 같이 “Add Credit Card Details” 버튼을 클릭하면 신용 카드 상세 정보 입력 폼이 출력됩니다.

아래 이미지와 같은 신용 카드 정보를 입력합니다. 여기서 입력한 신용카드 정보는 사용자 인증 용도로만 이용됩니다. Free Tier 사용과 관련된 어떠한 결재도 발생하지 않습니다.

신용 카드가 등록되면 다음과 같이 1달러 결재와 1달러 취소 트랜잭션이 실행됩니다. 이 결재와 취소 과정을 통해서 신용 카드 인증이 완료됩니다. Free Tier에 등록된 신용카드 정보는 사용자를 인증한 다음에는 어떤 결제에도 이용되지 않습니다.

신용 카드 등록이 완료한 후, 다음 그림과 같이 오라클 클라우드 사용 동의 체크 박스를 체크하고 “Complete Sign-up” 버튼을 클릭하면 모든 사용자 등록 절차는 완료됩니다.

사용자 등록 절차가 완료되면 다음 이미지와 같이 사용자 생성 중 페이지가 출력됩니다. 이 사용자 생성 페이지는 약 1분 정도 후에 오라클 클라우드 로그인 페이지로 자동 이동됩니다.

앞에서 입력한 로그인 E-mail과 패스워드를 입력하고 “Sign-in” 버튼을 클릭하면 Oracle Cloud에 로그인합니다.

사용자 정보를 확인하고 로그인이 완료되면, Oracle Cloud 메인 페이지에서 Free Trial과 Always Free가 활성화된 것을 확인할 수 있습니다.

요약

Oracle Cloud Free Tier는 기존 Oracle Cloud Free Trial을 확대한 프로모션입니다. Oracle Cloud Free Tier는 Oracle Cloud Free Trial과 Always Free로 구성됩니다.

Oracle Cloud Free Trial은 30일간 300달러 규모로 다양한 오라클 클라우드의 IaaS(VM, Bare Metal 서버, Function, Kubernetes)와 PaaS(Datbase, Kafak, NoSQL, CI/CE 등) 서비스를 이용할 수 있는 권한을 제공합니다. 추가로 Oracle Cloud Free Tier 사용자는 Always Free를 통해서 Autonomous Database인 ATP와 ADW 인스턴스를 각각 1개씩 그리고 VM 2개를 기간 제약 없이 계속 사용할 수 있습니다. 또한 Oracle Cloud Free Tier 사용자는 Always Free 프로모션을 통해서 Autonomous Database와 VM 이외에도 지정된 규모의 스토리지, 네트워크 트래픽, Load Balancer, 모니터링 서비스 및 알람 서비스도 기간 제약 없이 계속 이용 가능합니다.

Oracle Cloud Free Tier 계정 등록에는 개인 메일, 핸드폰 번호, 신용 카드 정보가 필요하며, 계정 등록에 사용되는 정보는 계정 등록과 사용자 인증 용도로만 사용됩니다. 특히 계정 등록에 사용된 신용 카드는 사용자 인증 용도로만 사용됩니다. Oracle Cloud Free Tier와 관련된 어떤 추가 결제도 발생하지 않습니다.
