오라클 클라우드와 AWS의 주요 서비스 이름 매핑

November 08, 2021 Cloud

오라클 클라우드(OCI, Oracle Cloud Infrastructure)와 AWS의 서비스를 비교하다 보면 비슷한 기능을 제공하는 클라우드 기능이 각각 다른 고유 서비스 이름으로 명명되기에 혼동되는 경우가 많습니다. AWS 혹은 OCI에 유사 서비스 이름을 정리하면, 특정 클라우드 서비스에 익숙한 사용자가 다른 서비스를 빠르게 이해하는데 도움이 될 것 같습니다.

이번 페이지에서는 AWS와 OCI 클라우드 서비스의 유사 기능 이름을 매핑하여 정리했습니다. 추가로 OCI를 기준으로 기능을 소개하는 페이지 링크를 정리했습니다. OCI 공식 매뉴얼의 「개요」 페이지 주소를 연결했습니다.

이 문서는 신규 서비스 및 기존에 빠진 서비스 매핑을 최신 버전으로 유지하기 위하여 지속적으로 업데이트해 나갈 예정입니다. 잘못된 서비스 이름 매핑이나 빠진 정보가 있다면 댓글로 남겨 주시면 업데이트 하겠습니다. [최종 업데이트: 2021년 11월 8일]
