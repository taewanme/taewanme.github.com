오라클 클라우드 서비스 아이콘 정리(2022.02)

February 22, 2022 Cloud

2022년 2월 오라클이 배포한 OCI(Oracle Cloud Infrastructure, 오라클 킄라우드) 서비스 로고 아이콘 이미지를 정리했습니다. 이 페이지에서 사용하는 OCI 서비스 아이콘은 2022년 2월 현재 배포된 OCI 서비스 공식 로고 이미지의 최신 버전입니다. OCI 서비스 공식 아이콘 이미지에 대하여 대상 OCI 서비스 이름, 서비스/기술 요약, 관련 서비스 페이지 링크 및 메뉴얼 페이지 정보를 정리합니다. 일관된 문서화와 정확한 정보 전달을 위하여 이 로고를 사용하여 문서화하고 콘텐츠를 제작할 계획입니다.

관련 자료는 다음 링크로 다운로드 받으실 수 있습니다.

  • 아이콘 이미지 PNG/SVG 전체 다운로드: https://docs.oracle.com/en-us/iaas/Content/Resources/Assets/OCI_Icons_PNG_SVG.zip
  • OCI 표준 스타일 파워포인트 파일: https://docs.oracle.com/en-us/iaas/Content/Resources/Assets/OCI_Icons.pptx
