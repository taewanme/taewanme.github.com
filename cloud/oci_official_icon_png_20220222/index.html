오라클 클라우드 서비스 공식 로고 이미지 모음: 2022.02

February 22, 2000 Cloud

2022년 2월 오라클에서 배포한 OCI(Oracle Cloud Infrastructure, 오라클 킄라우드) 공식 서비스 로고 이미지를 정리했습니다. 2022년 2월 현재 공식 OCI 서비스 공식 로고 이미지의 최신 버전입니다. 이 페이지를 통해서 OCI 공식 로고를 찾는 수고를 덜었으면 합니다. 향후 문서화를 진행할 때 정확한 정보 전달을 위하여 이 로고를 사용하여 문서화하고 컨텐츠를 제작하겠습니다.

관련 자료는 다음 링크로 다운로드 받으실 수 있습니다.

  • 아이콘 이미지 PNG/SVG 전체 다운로드: https://docs.oracle.com/en-us/iaas/Content/Resources/Assets/OCI_Icons_PNG_SVG.zip
  • OCI 표준 스타일 파워포인트 파일: https://docs.oracle.com/en-us/iaas/Content/Resources/Assets/OCI_Icons.pptx

Autonomous Transaction Processing(ATP), Autonomous Data Warehouse(ADW), Oracle APEX Application Development, Autonomous JSON Database의 네 가지 워크로드 유형을 지원합니다. 별도 하드웨어 구성 및 소프트웨어 설치가 필요 없습니다. 프로비저닝 후에는 가용성이나 성능에 영향을 주지 않고 언제든지 데이터베이스의 CPU 코어 수 또는 스토리지 용량을 확장할 수 있습니다.
