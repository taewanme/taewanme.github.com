    OCI LiveLogs

    1. LiveLog란?

    LiveLogs는 유튜브, 웹문서, 깃헙(Github), 슬라이드쉐어(Slidesahre)를 구조화하여 정보와 함께 데모를 제공하는 구조화된 컨텐츠입니다. LiveLog 형식으로 작성된 콘텐츠는 동영상을 통해서 개념와 데모 절차를 정보를 전달하고, 해당 동영상이 참조하는 문서를 이용하여 실습을 쉽게 따라할 수 있도록 구성됩니다. 위 동영상을 통해서 LiveLog의 의미를 확인할 수 있습니다.

    2. OCI(Oracle Cloud Infrastructure)

    오라클 클라우드는 2016년 하반기에 클라우드 서비스를 시작했습니다. 오라클 클라우드는 탄력적이고 유연한 자원 관리의 특징을 갖는 퍼블릭 클라우드입니다. 이런 디자인 원칙을 강조한 오라클 2세대 클라우드 IaaS 서비스를 OCI(Oracle Cloud Infrastructure)라고 합니다. 2020년 오라클 클라우드의 공식 브랜드 명은 Oracle Cloud에서 OCI(Oracle Cloud Infrastructure)로 변경됐습니다.

    3. OCI Livelogs

    OCI 사용자를 위하여 LiveLog 형식으로 OCI 사용자 가이드를 제공하는 것을 OCI Livelogs라고 합니다. OCI Livelogs는 youtube과 웹사이트로 구성됩니다.
