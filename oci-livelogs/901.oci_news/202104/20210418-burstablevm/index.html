    Compute: Burstable Instance

    2021년 4월 13일에 OCI Compute VM의 새로운 인스턴스 형태인 버스터블 인스턴스(Burstable Instance, 순간 성능 확장 가능 인스턴스)를 공개했습니다. 이와 관련 정보의 원본은 아래 링크에서 확인할 수 있습니다. 버스터블 인스턴스는 특정 순간에 성능을 확장 할 수 있는 저렴한하고 비용 효율적인 인스턴스 형태입니다.

    • [2021.04.13:Release Note]Burstable compute instances for small or low-utilization applications
    • [2021.04.14:Oracle Blog]Run your low-CPU workloads more cost-effectively with burstable VMs
    Burstable compute instances for small or low-utilization applications Oracle Blog: Run your low-CPU workloads more cost-effectively with burstable VMs

    OCI 새 소식은 OCI Release Notes에서 발표한 OCI 신규 업데이트 내용을 정리하는 섹션입니다. OCI 신규 기능 개선의 빠른 정보 전달을 목표로 합니다.

    비용 효율적인 성능 순간 확장형 인스턴스

    버스터블 인스턴스는 워크로드가 필요한 시점에 CPU 성능을 바로 확장하는 컴퓨트 인스턴스입니다. 일반적으로 유휴 상태이거나 가끔 사용량이 급증하는 경우 CPU 사용률이 낮은 시나리오를 위해 설계되었습니다. 버스터블 인스턴스는 가끔 사용량이 급증하면서 일반적으로 CPU 사용률이 낮은 워크로드에 적용하면 효과적인입니다.

    주기적이고 간혈적으로 CPU 사용량이 증가하는 다음과 같은 업무네 적용하면 클라우드 비용 최적화 잇점을 얻을 수 있습니다
