    OCI Docs

    OCI(Oracle Cloud Infrastructure)는 오라클이 제공하는 클라우드 서비스입니다. OCI Docs는 오라클 클라우드 입문자와 사용자를 위하여 OCI 관련 문서를 정리하는 페이지입니다. OCI는 매주 신규 기능을 공개하고 있습니다. OCI 서비스 변경에 맞춰 다양한 서비스에 대한 문서를 정리하겠습니다.

    OCIDocs는 다음과 같은 문서를 만들고 지속적으로 업데이트하여 가능한 최신 상태를 유지할 예정입니다.
