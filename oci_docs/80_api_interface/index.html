    OCI REST API 인터페이스

    Oracle Cloud Interface(OCI)는 모든 기능을 제어하는 REST API로 제공합니다. 이 REST API를 근간으로 다양한 인터페이스가 제공됩니다.

    OCI의 모든 기능은 API로 제어할 수 있습니다. OCI를 관리하고 제어하는 API는 여러 가지 형태로 제공하고 있습니다.

    SDKSoftware Development Kits : OCI 서비스와 연계하여 애플리케이션을 개발할 때 사용할 수 있는 SDK를 언어별로 제공합니다. SDK, 관련 문서, 코드 샘플 등을 제공합니다. SDK for Java Python SDK Ruby SDK Go SDK CLICommand Line Interface : OCI 서비스 제어를 명령행에서 할 수 있는 CLI를 제공합니다. Tools : 여러가지 DevOps 에서 OCI와 연계할 수 있도록 플러그인을 제공하고 있습니다. Terraform, Ansible, Chef 등등 REST API : OCI를 제어하는 API로 HTTPS을 기반으로 하는 REST API를 제공합니다.
